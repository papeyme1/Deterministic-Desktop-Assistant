# Deterministic Desktop Assistant
I'm against AI being used for everything as of late, however some voice assistant functions can be pretty useful. I want to make an assistant you can call with the push of a button that doesn't use AI for everything. Due to this limitation, unique functions will have to be coded in on the fly.
#
The assistant should be tailored perfectly to whoever is using it as the user is in charge of programming their own functions. I plan on making a GUI application/dashboard for the assistant where the functions can be updated and customization can be done outside a code editor for less tech savvy users.
